# Islamic Center of Moscow — Community Projects

A public transparency and community support platform for the Islamic Center of Moscow.

This website was created to help the community clearly see how donations are being used to maintain, improve, and support the Islamic Center. Community members can follow active projects, fundraising progress, repair updates, expenses, timelines, and project photos in one place.

The goal of this platform is to strengthen trust, accountability, and communication between the Islamic Center and the community by making project information openly accessible and easy to follow.

**Live Website:** https://icmoscow.com

---

## What You Can Find on the Website

- Active and upcoming community projects
- Public fundraising progress
- Donation and expense transparency
- Repair and maintenance updates
- Before and after project photos
- Community activities and programs
- Project timelines and milestones

The platform is designed to help donors and community members better understand:
- what projects are currently being worked on
- why fundraising is needed
- how donations are being used
- what progress has been completed

---

## Community Transparency

Every project page includes public information related to:
- fundraising goals
- donations received
- expenses and receipts
- project progress updates
- repair and maintenance photos

The website aims to provide a simple and organized way for the community to stay informed and engaged.

---

## Current Focus

The current phase of the platform focuses on:
- building transparency tools
- organizing project information
- sharing updates with the community
- improving visibility into fundraising and maintenance efforts

Future improvements may include:
- easier project submissions
- automated update workflows
- volunteer coordination
- expanded community announcements

---

## Tech Stack

This project is built using:
- Vue 3
- TypeScript
- Tailwind CSS
- Google Sheets
- GitHub Pages

The platform intentionally uses a lightweight architecture to keep maintenance simple and sustainable.
